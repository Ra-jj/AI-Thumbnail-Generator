-- Developed a full-stack AI Thumbnail Generator application using the MERN stack (MongoDB, Express.js, React.js, Node.js) with TypeScript and secure user authentication. The application allows users to register, log in, and log out to access AI-powered thumbnail generation features. Implemented JWT for token-based authentication and bcrypt for password hashing to ensure data security. The frontend was built with React.js and TypeScript, featuring a responsive user interface, dynamic rendering, and seamless API integration. On the backend, RESTful APIs were developed using Express.js and Node.js with TypeScript, and MongoDB was used for efficient data storage and user management.
